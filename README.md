# ChatGPT-s-note-on-Erdos451
This repo contains a paper, written by ChatGPT, that improves a bound on Erdos Problem #451, which concerns consecutive integers free of certain prime factors. This paper was then rewritten from scratch, and the human-generated result can be found on the arXiv. 

W. van Doorn and Q. Tang, Consecutive integers free of certain prime factors. arXiv:2606.19863 (2026).

The repo also contains a Lean formalization (obtained by Aristotle) of the main theorem that we prove.
